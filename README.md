
  # SANJEEVANI - AI powered Healthcare Assistant
   An AI-powered healthcare chatbotthat delivers personalized medical guidance, medicine reminders, vaccination tracking, and preventive health awareness to improve accessibility and health literacy across rural and urban communities.

  # Key Features:
  • AI-Powered Healthcare Chatbot providing reliable, verified medical guidance 24/7
  • Smart Medicine Reminder Systemwith multi-drug scheduling and adherence tracking
  • Automated Vaccination Schedule Management with personalized alerts in collaboration with health authorities
  • AI-Based Symptom Guidance & Safe Triageto suggest basic care and recommend when to visit a doctor
  • Regional Language & Voice Supportfor rural and low-literacy populations
  • Low-Bandwidth & Multi-Platform Access(Telegram, WhatsApp, SMS, mobile app compatibility)
  • Preventive Healthcare & Hygiene Awareness Campaignsto reduce misinformation 
  • Nearby Healthcare centre locations available all the time for emergency support

  # Innovation & Uniqueness
   Hybrid AI System–Combines ML disease prediction + LLM reasoning using Google Gemini API
   End-to-End Healthcare Flow–Predic on → Precau ons → Nearest Hospital → Medicine Reminders
   Multilingual Support–Removes language barriers for wider accessibility
   Microservice Architecture–Scalable and production-ready
